# Fashiostreet App

Full-Stack E-Commerce & Food Delivery Platform

🚀 Built from scratch with complete ownership across frontend development, backend APIs, database design, payment integrations, deployment workflows, and production support.

---

## Overview

Fashiostreet is a full-stack eCommerce and food delivery platform designed to provide seamless online ordering, product browsing, secure payments, and real-time customer experiences.

The project was built as a production-focused application with emphasis on:

* Fast and responsive user experience
* Scalable backend architecture
* Secure payment workflows
* Mobile-first experience
* End-to-end product ownership
* Real-world production deployment

This project represents one of my early startup-building experiences where I handled the complete product lifecycle independently.

## Architecture Overview

```text
Mobile App (Cordova)
        ↓
REST APIs (Laravel Backend)
        ↓
MySQL Database
        ↓
Payment & Notification Integrations
```

The application was designed with a modular backend structure and API-driven communication between frontend and backend systems.
